# Experiment Results

Each subfolder = one run. Name it descriptively.

| Folder | Config |
|--------|--------|
| `baseline/` | No delay, no loss |
| `delay_10_10/` | thermal=10ms, imagery=10ms |
| `delay_20_80/` | thermal=20ms, imagery=80ms (asymmetric) |
| `delay_50_50/` | thermal=50ms, imagery=50ms |
| `delay_100_100/` | thermal=100ms, imagery=100ms |
| `loss_1_1/` | thermal=1% loss, imagery=1% loss |
| `loss_5_5/` | thermal=5% loss, imagery=5% loss |
| `hz_1/` | SEND_HZ=1 |
| `hz_5/` | SEND_HZ=5 |
| `hz_10/` | SEND_HZ=10 |

Each folder contains:
- `fusion_log.csv` — per-event fusion decisions
- `latency_log.jsonl` — full latency breakdown
- `plots/` — generated by analyze_latency.py
